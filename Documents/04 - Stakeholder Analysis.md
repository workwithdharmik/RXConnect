## 3. Stakeholder Engagement Plan

| Stakeholder | Communication Method | Frequency | Key Information Shared | Owner |
| --- | --- | --- | --- | --- |
| CEO / Founder | Executive Steering Meeting | Bi-weekly | Milestone status, risks, budget | PM |
| Doctors (End Users) | User Workshop + UAT Session | Monthly + Pre-launch | Feature demos, feedback sessions | BA + PM |
| Development Team | Sprint Planning + Daily Standup | Daily / Bi-weekly | User stories, technical clarifications | PM + BA |
| Pharmacy Partners | Email + Integration Workshop | Monthly | API spec, integration timelines | BA |
| Receptionists | Training Sessions | Pre-launch + Post-launch | Feature walkthroughs, user guides | BA |
